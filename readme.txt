Есть список директорий, каждая из которых может иметь вложенные поддиректории, неизвестно насколько большой вложенности.
В директории любой вложенности может находится файл count, который сдержит одно число и ничего больше.
Нужно пройтись по всем директориям и вернуть сумму всех чисел из файла count (файлов count может быть много)

Результатом задачи является вывод одного числа.
